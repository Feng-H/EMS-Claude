# Project Instructions

## Local Development
- **No Docker**: Do NOT use Docker for local development. Run components (Go backend, Vite frontend) directly using their native tools.
- **Backend**: Navigate to the `backend/` directory and run `go run main.go`. Ensure local PostgreSQL and Redis services are accessible as per `backend/config/config.yaml`.
- **Frontend**: Navigate to the `frontend/` directory and run `npm run dev`.

## Workspace Conventions
- Follow the guidelines in `CLAUDE.md` for coding standards and workflows.
